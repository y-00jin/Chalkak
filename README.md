# 짤깍
### 📑프로젝트 개요

- 원하는 사용자와 여행 장소 및 가고 싶은 다양한 장소를 기록하고 공유할 수 있는 웹 플랫폼
<br><br>
### 💻 프로젝트 기능

- 사용자 연결
    - 연결 코드를 통해 다른 사용자와 연결
    - 연결된 사용자간 동일한 지도서비스 제공

- 지도서비스
    - Kakao API를 이용한 지도, 심볼, 장소 조회 기능

- 장소 정보 관리
    - 즐겨 찾기를 통해 장소 정보 저장

- 손 메모
    - 지도 위에 그림을 그릴 수 있는 기능
    - 경로를 그려 표시하거나 메모
